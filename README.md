# Hey, I'm Jadan 👋

I'm a software engineer from Jamaica. I build agentic AI systems, local-first tools, and the infrastructure that keeps them reliable.

I've worked across backend engineering, QA automation, cloud infrastructure, AI inference, and multi-agent workflows. Right now, I'm especially interested in how agents remember, make decisions, ask for approval, and interact with real systems without turning the whole thing into chaos.

Most of my public projects started with "I wish this existed" and got slightly out of hand.

## What I'm building

- [**SpectreOS**](https://github.com/In-sp3ctr3/spectreOS) is a safety-first personal AI operating shell with durable state, approvals, scheduling, memory, and real-world integrations.
- [**MemPR**](https://github.com/In-sp3ctr3/memPR) gives durable AI memory a pull-request workflow so agents and humans can review what gets remembered.
- [**Clack**](https://github.com/In-sp3ctr3/clack) is a native, local-first macOS clipboard manager with sensitive-content detection and Safe Paste.
- [**Paypa**](https://github.com/In-sp3ctr3/Paypa) reconstructs a trustworthy personal-finance ledger from fragmented bank emails and statements, starting with Jamaican banking workflows.

I usually reach for Python, TypeScript, Swift, Next.js, PostgreSQL, Playwright, Kubernetes, and Terraform. The stack changes from project to project. I care more about whether a system can explain what it did and fail safely when reality gets messy.
